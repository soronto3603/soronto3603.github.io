Mysql 데이터 export 하기

24 AUG 2018 • 1 min read

MySql 안의 데이터 .sql 파일로 export 하기

Export

mysqldump -u[user] -p[pw] [dbname] > [filename].sql

Import

mysql -u[user] -p[pw] [dbname] < [filename].sql
